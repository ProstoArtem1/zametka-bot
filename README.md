# 📝 Notes Telegram Bot

Телеграм-бот для хранения личных заметок.

## Возможности

- ✏️ Создание заметок
- 📚 Просмотр заметок
- 🗑 Удаление заметок
- 📊 Статистика пользователя
- 💾 Хранение данных в SQLite
- 🔄 FSM состояния
- 🎛 Inline-кнопки
- 📝 Логирование действий

## Команды

| Команда | Описание |
|----------|----------|
| /start | Запуск бота |
| /help | Инструкция |
| /list | Мои заметки |
| /delete | Удалить заметку |
| /stats | Статистика |

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python bot.py
```
